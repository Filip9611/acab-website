## Design-Richtlinien
- Farbschema: Schwarz/Weiss monochrom, harte Kontraste
- Stil: Industrial, rau, Werkstatt-Ästhetik
- Logo: Serif-Schrift "ACAB" mit Sans-Serif "ALL CARS · ALL BIKES"
- Hero: Video-Text-Masking-Effekt (CSS background-clip: text)
- Animationen: GSAP ScrollTrigger für Sektions-Übergänge
- Typografie: Serif für Headlines, Sans-Serif für Body
- Keine bunten Farben — nur Schwarz, Weiss, Graustufen

## Tech-Stack
- Next.js 15 (App Router, TypeScript)
- Tailwind CSS v4
- GSAP (ScrollTrigger, SplitText)

## Regeln
- Mobile-first responsive
- Performance: Core Web Vitals optimiert
- SEO: Meta-Tags, Open Graph, JSON-LD Schema (LocalBusiness)
- Accessibility: WCAG 2.1 AA
- Sprache: Deutsch (Schweiz)
- Video: WebM + MP4 Fallback, lazy-loaded, autoplay muted loop
